# Model Notes

## Voice

- Primary engine: OmniVoice
- Input: reference voice sample and text
- Output: generated audio

## Avatar

- Primary engine: SadTalker
- Input: portrait image and generated audio
- Output: talking head video

## Refinement

- Optional: Wav2Lip
- Use only when lip alignment needs a post-process pass
